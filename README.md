# MacLander
MacLander lets you instantly view your Mac's status with Time, Ram, Calendar, CPU, Battery, AND lets you open your Mac's apps [COMING SOON: File Browsing]
Website: gellybob.github.io
## Installation

To download and open the installer instantly via your Mac Terminal, copy and paste this command:

```bash
curl -L -o ~/Downloads/MacLander.Installer.dmg "https://github.com" && open ~/Downloads/MacLander.Installer.dmg
```
